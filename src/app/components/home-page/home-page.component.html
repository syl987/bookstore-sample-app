Tech-Stack

These technologies are the backbone of the application.

Angular Main Framework

Clean and consistent application structure. Heavily using rxjs' Observables and SCSS for styling.
NGRX State Management, Logic

Application State and View fully decoupled! View all events in the browser console in real-time.
RXJS Reactive Extensions

Complex reactive processing of API requests and application data for display purposes.
Firebase Backend Services

Using authentication, NoSQL realtime database, file storage and web hosting services.
Material Angular Components

Native Angular components with customized theming, typography and general appearence.
Bootstrap Grid, Helpers, Typography

Consistent global syntax for grid, helpers and typography. Using its variables as center piece.
TypeScript JavaScript Extended

Strictly configured for null-safety, safe variable initialization and reliable use of types.
SCSS CSS Extended

Leverage variables and CSS generation features, global and component-scoped.
Eslint Code Linting

Semi-automated code quality tooling with extensive custom configuration and plugins.
Prettier Code Formatting

Fully automated opinionated code quality tooling for appearence and consistency.
Git Version Management

Structured and persistent development progression using branching, stashing, rebasing.
Github Code Management

Transparent development experience using pull requests and branch safety options.

Technical Features

Authentication

Browse anonymously, login to interact with existing books or create your own.
Internationalization

Fully translated application content with respective locale formatting.
Responsive Design

The semi-dynamic application layout fully supports all common device sizes.
NoSQL Database

Any changes by the user are persistently saved using Firebase realtime database.
File Upload

Select, crop and upload image files for display alongside with regular data.
Forms

A variety of form fields with consistent offline and on-response validation.
External API

Find your favourite books with Google Books API and provide custom data.
